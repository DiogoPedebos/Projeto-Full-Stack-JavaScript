------ Projeto para gerência de Tarefas ------- 
-----------------------------------------------
Para iniciar o código:

    abra a pasta backend no bash e inicie com:
        npm start

    abra a pasta frontend no bash e inicie com:
        npm start

-----------------------------------------------
backend
- Configurar server para receber todos os valores
- Realizar configuração de API para comunicação com base de dados

front
- Criar condição para que seja exibido todos os campos para adição
apenas quando clicado em adicionar tarefa
- Criar página de login
- Implementar nível de acesso
- Editar estilo do site
