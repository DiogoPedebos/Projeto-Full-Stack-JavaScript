Para iniciar o código:

    abra a pasta backend no bash e inicie com:
        npm start

    abra a pasta frontend no bash e inicie com:
        npm start

-----

- Alterar os atributos do objeto tarefa para: 
    título;
    descrição;
    status (pendente, em andamento, concluída); 
    data de criação;
    data de conclusão.
